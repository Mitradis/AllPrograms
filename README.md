# AllPrograms

Программа-менеджер ярлыков для приложений, файлов и папок. Поддерживается свое название ярлыка, дополнительная команда запуска, добавление\удаление ярлыков прямо в программе, изменение размера окна (автоматическое перестраивание), безрамочный вид, свой ini парсер. Все настройки и ссылки сохраняются в ini файл. Если выбранный объект-ярлык находится на одном диске с программой, то путь да файла\папки становится относительным, благодаря этому можно использовать на съемных носителях, либо в рамках одной перемещаемой папки.
